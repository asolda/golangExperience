
******* Lexer example for PASCAL **********


La cartella contiene i seguenti files:

    pascal.jflex -- the JFlex lexer specification for Pascal (Attenzione: in Eclipse vuole estensione .lex)

    PascalLexerTest.java -- a simple testing program, 

    sym.java -- token definitions, generated by a CUP specification (not present here)

    lexer-test.p -- a Pascal program to test the lexer

    raw-pascal-tokens.txt -- a bunch of raw tokens to test the lexer; works just fine this way

    make.csh -- UNIX shell script to compile and run the Pascal lexer example; mostra quali comandi lanciare da linea di comando (se non si usa Eclipse)


Lancio di flex (ed anche javacup) tramite Eclipse: basta fare una qualsiasi modifica al .lex e salvare il file.
